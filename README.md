# Swiggy Sales Analysis Dashboard

## Project Overview
This project analyzes Swiggy sales data using Power BI.

The dashboard helps understand:
- Total Sales
- Top Cities
- Top Restaurants
- Order Trends
- Customer Ratings

## Tools Used
- Power BI
- Excel

## Dataset
The dataset contains:
- Restaurant names
- City
- Orders
- Ratings
- Revenue

## Dashboard Insights
- Identified top-performing restaurants
- Compared city-wise sales
- Analyzed customer ratings
- Found peak order periods
